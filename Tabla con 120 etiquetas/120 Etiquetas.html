< html >

HTML es un markup language, lo que significa que esta escrito con codigos que puede leer una persona sin que sea necesario compilarlo primero. En otras palabras, el texto en una pagina web eta macado con estos codigos para dar intrucciones al navegador web sobre como mostrar el texto.

< meta >

Son un tipo de etiqueta incorporada en la cabecera de una pagina web, que permite a los navegadores u otros programas que puedan valerse de informacion suministrada por la web. El proposito de este tipo de elementos es incluir informacion de referencia sobe la pagina como puede ser: autor,fecha, descripcion, palabras claves, entre otros. Estas etiquetas tambien suelen especificar cierta informacion tecnica util para que el navegador pueda mostrar la pagina correctamente.

< title >

Esta etiqueta sirve para ponerle un titulo al navegador de nuestra pagina.

< head >

Conocida en espanol como cabecera, es la parte que no se muestra en el navegador en el momento de cargar la pagina. Esta etiqueta puede contener varios hijos que otorgan informacion sobre el sitio en cuestion. Algunos de estos son la etiqueta title define el titulo de la pagina, los enlaces a css o de faviconos, y cualquier otro tipo de metadato, como es el tipo de caracter empleado en la pagina, quien la realizo, su descripcion,entre otros. Los siguiente elemetos de HTML pueden ir dentro de la etiqueta head:

  • title

  • style

  • base

  • link

  • Meta

  • script

  • noscript
<header>

Es normalmente utilizada como el encabezado de una seccion en ellos pueden encontrase diversos elementos como vinculos,logos,tablas de contenido o algunas introducciones.

< body >

En esta etiqueta estara todo el cuerpo de la pagina web todo el contenido como imagenes, articulos, funcionalidades ect..

<link>

Es una etiqueta que crea un hiperenlace y normarmente enlaza externamente a otra pagina o documento en una web diferente o internamene a otros recursos de la web.

<footer>

Representa un pie de pagina para el contenido de la seccion mas cercano o el elemento raiz de seccion

<figure>

Representa contenido independiente, a menudo con un titulo. Si bien se relaciona con el flujo pricipal, su posicion es independiente de este. Por lo general, se trata de una imagen, una ilustracion,un diagrama,un fragmento de codigo al que hace referencia en el texto principal

<table>

Esta etiqueta nos sirve para crear un tabla Las tablas mas sencillas se definen con 3 etiqueta table para la tabla , tr para la filas,td para las columnas

<input>

El elemento input representa un campo de datos tipado que normalmente se asocia con un control que permite a los usuarios editar su valor. Este elemento es capaz de proveer muchos tipos diferentes de campos, de acuerdo al valor que presente en el atributo type.

<center>

Esta etiqueta te permite central el texto desde el mismo html sin la necesidad d de usar css.
